SECTION 1: MASTERING THE LINUX TERMINAL
Lesson 2: Basic commands

[echo]
Prints Command Line args to Standard Output

[date]
Shows the current date and time

[cal]
Displays a calendar

[cat]
Sticks files together and writes joined file to Standard Output
Good for viewing the contents of one file

Examples:
CMD: echo Hello World
OUT: Hello World

CMD: date
OUT: Mon 30 Nov 2020 03:28:23 PM UTC

CMD: cal
OUT:
    November 2020
